API Timestamp Microservice

Description

A server-side application that allows you to process
the timestamp parameter. This is my final solution of the
FreeCodeCamp Timestamp Mircoservice challenge.

Technology Stack

Rest API • ExpressJS • TypeScript
NodeJS • Morgan • Nodemon • TS-Node

Example Usage

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000

Example Output

{"unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT"}
